Design Lib Inspector — UX/UI-аудит макета по 227 правилам за один клик.

Проверяет:
— Контраст WCAG 2.2, тач-зоны (iOS 44, Android 48)
— Типографическую шкалу, интерлиньяж, длину строки
— Консистентность radius/обводок/теней/палитры
— Focus/disabled, метки полей, закон Хика
— Composition, иерархию, формы, dark mode (через Claude Vision)

Как работает:
— 29 детерминированных детекторов за <5 мс локально
— AI-аудит (97 правил): Opus 4.7 с deep-thinking. Только BYOK, ключ хранится в figma.clientStorage
— 3 батча параллельно, fuzzy-матч слоя для jump-to-layer

Результат:
— Score 0–100 с вердиктом (Критично → Отлично)
— Severity + priority P1–P5
— Группировка дублей, кросс-страничный переход к слою
— Экспорт отчёта в Markdown

Не проверяет скрытые слои (visible=false, opacity=0). Интерфейс на русском.
