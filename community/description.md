Design Libs Inspector — аудит макета по 227 правилам за секунды.

Проверяет:
— Контраст по WCAG 2.2, размер таргетов (iOS 44, Android 48, Web 24)
— Типографическую шкалу, межстрочный, letter-spacing, длину строки
— Сетку 4/8 px, радиусы, обводки, тени, размер палитры
— Чистые #000/#FFF в тёмной теме, focus/disabled, метки полей
— Закон Хика (≤7±2 действий в группе)

Как работает:
— 22 детектора прогоняются локально за <500 мс
— Claude Vision (Haiku/Sonnet/Opus) проверяет иерархию и визуальный вес
— Ключ Anthropic — только у вас в Figma clientStorage, запрос идёт напрямую в api.anthropic.com. Без телеметрии и прокси.

Результат:
— Проблемы с severity, jump-to-layer и игнорами на уровне документа
— Оценка 0–100 и экспорт в Markdown
— «Правила» — контекстные подсказки для выделенного слоя
