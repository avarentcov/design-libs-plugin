# Community assets

Ассеты для публикации в Figma Community.

| Файл | Требование Figma |
|---|---|
| `icon.svg` | 128×128, SVG/PNG |
| `cover.svg` | 1920×1080 (экспортировать в PNG перед загрузкой) |
| `screenshots/*` | 6 изображений 1920×1080 PNG (добавить перед сабмитом) |
| `description.md` | ≤1000 символов RU |
| `tags.json` | 12 тегов |

## Экспорт SVG → PNG перед загрузкой

```bash
# требует установленный rsvg-convert или inkscape
rsvg-convert -w 128 -h 128 icon.svg -o icon.png
rsvg-convert -w 1920 -h 1080 cover.svg -o cover.png
```
